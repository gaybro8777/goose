\section{Program goose-renamehumanheaders}
The \texttt{goose-renamehumanheaders} changes the headers of FASTA or Multi-FASTA file to simple chrX by order, where X is the number.\\
For help type:
\begin{lstlisting}
./goose-renamehumanheaders -h
\end{lstlisting}
In the following subsections, we explain the input and output paramters.

\subsection*{Input parameters}

The \texttt{goose-renamehumanheaders} program needs two streams for the computation,
namely the input and output standard. The input stream is a FASTA or Multi-FASTA file.\\
The attribution is given according to:
\begin{lstlisting}
Usage: ./goose-renamehumanheaders [options] [[--] args]
   or: ./goose-renamehumanheaders [options]

It changes the headers of FASTA or Multi-FASTA file to simple chr$1 by order.

    -h, --help            show this help message and exit

Basic options
    < input.fasta         Input FASTA or Multi-FASTA file format (stdin)
    > output.fasta        Output FASTA or Multi-FASTA file format (stdout)

Example: ./goose-renamehumanheaders < input.fasta > output.fasta
\end{lstlisting}
